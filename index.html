Random Policy Baseline Results

Overview

This website presents baseline results for a random policy on robotic manipulation tasks used in the 16-831 course project. The purpose of this baseline is to illustrate the difficulty of the environments and provide a reference point for subsequent reinforcement learning experiments.

We evaluate a random agent that samples actions uniformly from the action space on three manipulation tasks simulated using the Franka Panda robot in the panda-gym environment.

Experimental Setup

  • Environments: PandaReach, PandaPush, PandaPickAndPlace (panda-gym)
  • Policy: Uniformly random actions
  • Runs: 3 independent runs per task
  • Episodes per run:20
  • Metric: Mean episodic return

Task success is determined using the environment’s built-in threshold-based success condition. This metric is used consistently across all experiments.

Quantitative Results

Task Run 1 Mean Return Run 2 Mean Return Run 3 Mean Return
PandaReach -7.371 -9.279 -7.636
PandaPush -8.545 -8.651 -8.566
PandaPickAndPlace -8.726 -8.366 -10.878

The random agent achieves low mean episodic returns across all tasks, with occasional success in simpler tasks such as reaching and near-zero success in more complex manipulation tasks. These results highlight the non-triviality of the environments.
